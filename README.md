# Brobotbro — Single-file Telegram Bot (JSON Storage)

## Files
- main.py (كل البوت في ملف واحد)
- requirements.txt
- Procfile
- railway.json

## ENV Variables (Railway Variables)
BOT_TOKEN
SUPER_ADMIN_ID
REQUIRED_CHANNEL
SUPPORT_USERNAME
DATA_DIR (default: data)
MIN_TOPUP (default: 15000)
MIN_WITHDRAW (default: 50000)
LOG_LEVEL (default: INFO)

## Run
python main.py
